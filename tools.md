# tools

this is a list of useful tools i use for my system and developing stuff. this is in addition to the already listed deps in [README.md](README.md)

## cli

here are tools for cli stuff

```sh
# ripgrep for searching file contents
pacman -S rg

# fuzzy finder for finding stuff in a list of things like files
pacman -S fzf

# zoxide for replacing cd
pacman -S zoxide

# vim for light editing
pacman -S gvim

# neovim my beloved
pacman -S neovim
```

## dev

here are tools for dev stuff

# gcc
